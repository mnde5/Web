# Team2 LMS Frontend

Vite + React frontend for the Team2 LMS module.

## Project structure

```text
.
├── docs/              # Notes and role/access documentation
├── public/            # Static assets served by Vite
├── src/
│   ├── App.jsx        # Top-level routes
│   ├── main.jsx       # Vite entry point
│   ├── components/    # Shared route/helper components
│   └── team2/         # Team2 feature pages, services, hooks, utils
├── index.html
├── package.json
├── tailwind.config.js
└── vite.config.js
```

## Commands

```bash
npm install
npm run dev
npm run build
```
